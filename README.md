# Directorio Valladolid

Directorio web de recursos de Valladolid, creado para la organización [Aldea Pucela](https://aldeapucela.org/).

## Tecnologías

- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS v4.
- **Build Tool**: Vite.
- **Datos**: Conectado a la API pública de NocoDB de Aldea Pucela.

## Desarrollo Local

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Arrancar servidor de desarrollo:
   ```bash
   npm run dev
   ```

## Características de la versión 0.2

- Diseño responsivo y moderno basado en eventos.aldeapucela.org.
- Soporte para Modo Oscuro y Claro.
- Integración en tiempo real con datos de NocoDB.
- Buscador por texto.
- Filtro por Categorías.
- Filtro "Solo Oficiales".
- Nube de Etiquetas interactivas y múltiples.
